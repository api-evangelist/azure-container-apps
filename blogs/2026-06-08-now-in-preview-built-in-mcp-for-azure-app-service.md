At Build 2026 last week, we announced the public preview of built-in MCP for Azure App Service . It does one thing, and it does it with almost no effort on your part: it turns a REST API you already host on App Service into a Model Context Protocol (MCP) server, so AI agents and assistants can call your API as a set of tools. No MCP code to write.
